# JavaScript Learned By Code With Harry

## Overview

This folder contains JavaScript code and projects that I learned through **Code With Harry's YouTube Tutorials**, PDF notes, websites, and other learning resources. It showcases my journey from beginner to advanced JavaScript concepts.

## Learning Path

### Beginner Level

- JavaScript Fundamentals
- Variables and Data Types
- Operators (Arithmetic, Logical, Comparison)
- Conditional Statements (if, else, switch)
- Loops (for, while, do-while)
- Functions and Scope
- Arrays and Array Methods
- Objects and Object Methods
- DOM Manipulation Basics
- Events and Event Listeners

### Intermediate Level

- ES6+ Features (Arrow Functions, Template Literals, Destructuring)
- Closures and Higher-Order Functions
- Asynchronous JavaScript (Callbacks, Promises)
- Async/Await
- Error Handling (try-catch-finally)
- Regular Expressions
- localStorage and sessionStorage
- JSON and API Interaction
- Classes and OOP Concepts
- Prototypes and Inheritance

### Advanced Level

- Advanced DOM Manipulation
- Event Delegation and Bubbling
- AJAX and Fetch API
- REST API Integration
- Module Systems (ES6 Modules, CommonJS)
- Functional Programming Concepts
- Performance Optimization
- Debugging Techniques
- Web APIs (Geolocation, Canvas, Web Workers)
- Build Tools and Bundlers (Webpack, Vite)
