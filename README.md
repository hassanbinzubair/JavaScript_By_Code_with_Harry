# JavaScript Learned By Code With Harry

## Overview

This Repository contains JavaScript code that I learned through **Code With Harry's** YouTube Tutorials, PDF notes, websites, and other learning resources. It showcases my journey from beginner to advanced JavaScript concepts.

## Learning Path

- JavaScript Fundamentals
- Variables and Data Types
- Operators (Arithmetic, Logical, Comparison)
- Conditional Statements (if, else, switch)
- Loops (for, while, do-while)
- Functions and Scope
- Arrays and Array Methods
- Objects and Object Methods
- DOM Manipulation Basics
- Events and Event Listeners
- ES6+ Features (Arrow Functions, Template Literals, Destructuring)
- Asynchronous JavaScript (Callbacks, Promises)
- Async/Await
- Error Handling (try-catch-finally)
- localStorage and sessionStorage
- Classes
- Prototypes and Inheritance

## Learning Resources

- 📄 **PDF Notes** - Study Materials
- 🌐 **Websites** - Reference Documentation
- 💻 **Hands-on Practice** - Project-based Learning

## How to Use This Repository

1. Browse through the files to find different JavaScript topics
2. Each file contains JavaScript code with examples and explanations
3. Open the HTML/JS files in your browser or run them in Node.js
4. Review the code comments for detailed explanations
5. Modify and experiment with the code to learn better
6. Use the browser console for testing and debugging
